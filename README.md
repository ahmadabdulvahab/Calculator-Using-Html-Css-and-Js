# Calculator

A simple and responsive calculator built using **HTML**, **CSS**, and **JavaScript**. The application performs basic arithmetic operations through an intuitive and user-friendly interface while demonstrating core front-end web development concepts.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Responsive interface
- Clean and simple design

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Structure

```
calculator/
├── index.html
├── style.css
├── script.js
└── README.md
```
